CSS Interview Concepts with Examples

1. What is box-sizing?

The box-sizing property defines how the total width and height of an element is calculated:

  • content-box (default): width and height include only content, padding and border are added outside.
  • border-box: width and height include content, padding, and border.

2. What is animation-delay?

The animation-delay property defines how long to wait before starting the animation on an element.

Animated Box

3. Which property do you use to merge cells vertically in a grid?

Use grid-row: span n; to make an element span across multiple rows.

4. Which property do you use to merge cells horizontally in a grid?

Use grid-column: span n; to make an element span across multiple columns.

5. Explain the CSS Box Model

The CSS box model consists of:

  • Content: The actual text or image
  • Padding: Space around the content
  • Border: Line around the padding
  • Margin: Space outside the border
Box Model Visual

6. What is the difference between display: none and visibility: hidden?

  • display: none removes the element from layout and does not reserve space.
  • visibility: hidden hides the element but still takes up space in the layout.
